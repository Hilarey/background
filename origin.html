background-origin

Свойство CSS background-origin задает область фонового позиционирования. Другими словами, это свойство устанавливает исходную позицию для изображения background-image.

Зависимости: необходимо установить background-repeat: no-repeat;

Поддержка браузерами
border-box
background-origin: border-box;
padding-box
background-origin: padding-box;
content-box
background-origin: content-box;
